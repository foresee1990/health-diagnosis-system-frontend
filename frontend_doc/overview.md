# 前端概述

## 一、系统简介

本系统为基于 AI 的在线健康问诊平台，用户可通过浏览器与 AI 医生进行实时对话问诊，并在问诊结束后生成 PDF 健康报告。

---

## 二、页面列表与功能说明

### 1. 登录页 `/login`

| 功能 | 说明 |
|------|------|
| 用户登录 | 输入用户名+密码，调用登录接口，获取并保存 JWT Token |
| 跳转注册 | 提供"去注册"链接 |
| 登录后跳转 | 登录成功后跳转至主聊天页 `/chat` |

---

### 2. 注册页 `/register`

| 功能 | 说明 |
|------|------|
| 用户注册 | 输入用户名、密码（可选邮箱），调用注册接口 |
| 注册成功 | 提示成功后跳转至登录页 |
| 表单校验 | 用户名 3-50 字符，密码 6-20 字符 |

---

### 3. 主聊天页 `/chat` 和 `/chat/:consultationId`

这是系统的核心页面，采用**三栏布局**：

```
+------------------+-------------------------------------+
|   顶部导航栏（64px，全宽）                             |
+------------------+-------------------------------------+
|                  |                                     |
|  左侧边栏        |  主聊天区域                         |
|  (260px)         |                                     |
|                  |                                     |
+------------------+-------------------------------------+
```

**左侧边栏功能：**

| 功能 | 说明 |
|------|------|
| 新建问诊按钮 | 弹出输入框填写主诉，调用创建接口后进入新会话 |
| 历史问诊列表 | 分页加载当前用户所有问诊，点击切换到对应会话 |
| 激活状态高亮 | 当前打开的会话在列表中高亮显示 |

**顶部导航栏功能：**

| 功能 | 说明 |
|------|------|
| Logo + 系统名 | 固定展示 |
| 用户头像 | 点击显示下拉菜单：我的信息、退出登录 |
| 退出登录 | 清除本地 Token，跳转 `/login` |

**主聊天区域功能：**

| 功能 | 说明 |
|------|------|
| 消息历史展示 | 进入会话时加载历史消息，区分用户消息（右对齐）和 AI 消息（左对齐） |
| 流式发送消息 | 用户输入内容后点击发送，调用 SSE 流式接口，AI 回复逐字打印 |
| 风险等级展示 | SSE 结束时接收 `riskLevel`，在聊天头部展示对应标签 |
| 结束问诊 | 提供"结束问诊"按钮，调用 PATCH 接口，会话进入 completed 状态 |
| 生成报告 | 会话 completed 后显示"生成报告"按钮，调用生成报告接口 |
| 下载报告 | 报告生成后提供下载链接（直接跳转后端文件接口） |

---

## 三、前端技术栈

| 技术 | 版本 | 用途 |
|------|------|------|
| Vue 3 | 3.x | 核心框架，使用 Composition API + `<script setup>` 语法 |
| Vite | 5.x | 开发构建工具，启动快、配置简单 |
| Vue Router 4 | 4.x | 前端路由管理 |
| Pinia | 2.x | 状态管理，比 Vuex 更简洁 |
| Axios | 1.x | HTTP 请求封装 |
| Element Plus | 2.x | UI 组件库，提供表单、按钮、对话框等组件 |

> 选型理由：以上技术栈是目前 Vue 生态最主流、文档最完善的组合，社区资料丰富，适合毕业设计使用。
